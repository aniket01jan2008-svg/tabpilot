# TabPilot

TabPilot is a focus workspace for students. It turns scattered study tabs,
links, notes, deadlines, and tasks into one clear action board.

Built for CodeStorm 2026 #2.

## Problem

Students often save useful material across browser tabs, PDFs, videos, notes,
chat messages, and todo lists. The information exists, but it is not organized
into what should be done today.

## Solution

TabPilot gives students a simple workflow:

1. Capture a link, note, deadline, or task.
2. Organize it into categories such as Focus, Research, Assignment, Coding,
   Deadline, or Later.
3. Convert everything into a three-task focus board.
4. Track completion and reduce context switching.

## MVP Features

- Responsive landing page and dashboard preview
- Quick capture interface
- Three-task focus checklist
- Workspace lanes for Focus, Research, and Later
- Student-specific positioning for assignments, exam prep, coding, and hackathon work
